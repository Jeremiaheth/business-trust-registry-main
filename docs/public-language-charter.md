# Public Language Charter

BTR-NG uses restrained public language because the public beta is a reviewable evidence layer, not an authority that can make final judgments about a business.

## Forbidden phrases

Do not publish these phrases in public-facing docs, templates, or user-visible copy:

- certified
- guaranteed
- verified by government
- official rating
- trusted business
- fraudulent

## Preferred phrases

Prefer the following phrases because they set the right expectation for a public beta:

- evidence dossier
- time-stamped verification event
- derived from published procurement data
- based on available verified evidence
- confidence indicates evidence completeness

## Practical rules

- Treat every score as decision support only.
- Explain when scoring is suppressed because a profile is under review, evidence is thin, or maintenance mode is active.
- Use source links, hashes, and timestamps instead of authority claims.
- Keep procurement signals framed as complementary evidence, not an official replacement rating.
