# Moderation Boundary

BTR-NG public moderation stays narrow in the public beta.

## Allowed public workflows

- claim a public business profile
- correct non-sensitive public metadata
- point to public links and hashes
- request fact-correction review for evidence sufficiency or metadata accuracy

## Disallowed public workflows

- uploading evidence files
- posting personal data
- filing third-party complaints about conduct
- asking the public repo to adjudicate disputes beyond fact correction

## Operator rules

- keep moderation actions tied to public-lane scope only
- reject submissions that include personal data or attachments
- move ambiguous or risky submissions out of the public lane
- preserve a clear audit trail of what changed and why
