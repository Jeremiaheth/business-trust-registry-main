# Code of Conduct

This project expects professional, respectful, and fact-based collaboration.

## Expected Behaviour

- Focus on technical issues and evidence.
- Respect contributor boundaries and privacy.
- Avoid personal attacks, harassment, or intimidation.

## Enforcement

Report incidents to the maintainers listed in the repository governance files once they are added. Project maintainers may remove content or restrict participation to protect contributors and the public safety posture of the project.
